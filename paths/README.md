# Course Paths

Six modules. Each takes 45–90 minutes. Run them in order the first time.

| # | Module | Pillar | Lab |
|---|---|---|---|
| 01 | Foundations | All | — |
| 02 | Context Management | 1 | `labs/lab-01-context` |
| 03 | Plan Mode | 2 | `labs/lab-02-plan-mode` |
| 04 | Externalization | 3 | `labs/lab-03-externalization` |
| 05 | Verification | 4 | `labs/lab-04-verification` + `labs/lab-05-break-the-hook` |
| 06 | Enterprise Hardening | All | `labs/capstone-spec-driven` |

Each module has:

- `README.md` — what you'll learn, why, and the exit criteria
- `exercises.md` — hands-on tasks with checkpoints
- A pointer to its lab directory (where applicable)

## How to use this

**Self-paced:** read the module README, do the exercises, run the lab, check yourself against the exit criteria. Move on when you can do all of them without notes.

**Instructor-led:** see `instructor/facilitator-guide.md` for timing, common questions, and the demo script.
